Python GUI programs made in Riveria class.

# Muistio / Notepad
Notepad clone made with tkinter.
Requires PIL, fpdf and pypdf.
Features:
- Basic notepad functionality.
- Saving and reading PDF files.
- Saving images to PDF file.
